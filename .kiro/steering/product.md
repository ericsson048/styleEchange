# Product: StyleÉchange

A C2C (consumer-to-consumer) fashion marketplace — think Vinted — where users can buy and sell second-hand clothing and goods.

## Core Features
- Product browsing with category filters and advanced search (size, brand, price, condition)
- Multi-step product listing form for sellers
- AI-assisted product description generation (via Genkit + Gemini)
- Checkout flow with shipping options
- Real-time buyer/seller messaging
- User profiles with listings, ratings, and order history
- Favorites / wishlist with heart icon toggle

## Design Language
- Primary: deep muted blue-grey (`#292E36`)
- Background: light cool grey (`#FAFBFC`)
- Accent: vibrant turquoise (`#25C1B1`) — used for CTAs and highlights
- Font: Inter (all text)
- Icons: lucide-react exclusively
- Products displayed in responsive grid (2 → 5 columns)
- Skeleton loaders for loading states
- Mobile filters in modals/sheet overlays
- Sticky navbar

## Locale
The app is in French. UI text, labels, and copy should be written in French.
