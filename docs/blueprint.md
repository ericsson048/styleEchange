# **App Name**: StyleÉchange

## Core Features:

- Product Discovery & Filtering: Browse product listings in a responsive grid, utilize a search bar with advanced filters (size, brand, price, condition), and navigate through categories to find items.
- Product Listing Form: A multi-step form allowing sellers to upload images, provide essential product details (title, description), assign categories, and set pricing for items they wish to sell.
- AI Product Description Assistant: A tool leveraging generative AI to assist sellers by suggesting detailed product descriptions and relevant keywords based on initial input and selected product categories.
- Secure Checkout Process: A streamlined checkout flow where buyers can review their order summary, select shipping options (including relay points), and complete secure payments for purchases.
- Real-time Messaging System: An integrated chat interface enabling real-time communication between buyers and sellers for inquiries, negotiation, and transaction coordination.
- User Profile Management: Public profile views for users to showcase their 'dressing' (listings) and view ratings, alongside a private section for managing their own product listings, orders, and personal account settings.
- Favorites / Wishlist: Allow users to 'favorite' products with a prominent heart icon, enabling them to save items to a personalized wishlist for later review.

## Style Guidelines:

- Primary color: A deep, muted blue-grey (#292E36) to be used for main text, primary headings, and functional UI elements, providing strong contrast on light backgrounds.
- Background color: A very light, cool grey (#FAFBFC) for the overall application background, offering a clean and neutral canvas that ensures legibility.
- Accent color: A vibrant turquoise (#25C1B1) reserved for calls-to-action, interactive elements, and key highlights, injecting a modern and engaging pop of color.
- Font selection: Use 'Inter' (sans-serif) for all text elements, including headlines and body content, to ensure a clean, contemporary, and highly readable user interface.
- Icon library: Integrate 'lucide-react' for all icons, maintaining a consistent, elegant, and crisp visual style throughout the application.
- Product display: Products are presented in a responsive grid layout, optimizing for visual browsing across various device sizes.
- Navigation bar: Implement a sticky header navigation bar that provides constant access to essential functions like search, filters, and the 'Sell' button.
- Mobile filters: On mobile devices, filters are presented within modal dialogs or slide-overs to ensure an intuitive and space-efficient user experience.
- Loading states: Utilize animated skeleton placeholders for images and content areas to improve perceived loading performance and user feedback.
- Interface transitions: Employ subtle and smooth transition animations for opening/closing modals, 'favoriting' actions, and navigation movements, enhancing interactivity without distraction.
